# Class Sheet: MessageOrigins

**Namespace:** Carebed.Infrastructure.Enums

---

## Purpose
Enumerates the possible origins of messages in the Carebed system. Used for tracking and routing messages based on their source module.

---

## Enum Definition
```
public enum MessageOrigins
```

## Values
- `Unknown = 0`
  - Default value for unspecified origins.

- `SensorManager`
  - Message originated from the Sensor Manager module.

- `ActuatorManager`
  - Message originated from the Actuator Manager module.

- `LoggingManager`
  - Message originated from the Logging Manager module.

- `SystemInitializer`
  - Message originated from the System Initializer.

- `DisplayManager`
  - Message originated from the Display Manager module.

- `AlertManager`
  - Message originated from the Alert Manager module.

- `NetworkManager`
  - Message originated from the Network Manager module.

- `EventBus`
  - Message originated from the Event Bus itself.
