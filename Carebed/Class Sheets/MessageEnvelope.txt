# Class Sheet: MessageEnvelope<T>

**Namespace:** Carebed.Infrastructure.MessageEnvelope

---

## Purpose
A concrete implementation of `IMessageEnvelope` that wraps a strongly typed payload and provides standardized metadata for messages in the Carebed system.

## Notes
- Used for encapsulating messages with metadata for event bus communication.
- Ensures payload is never null (enforced in constructor).
- Implements `IMessageEnvelope` for interoperability with non-generic code.
- Provides both strongly-typed (`T Payload`) and non-generic (`object Payload`) access to the payload.
- `MessageId` and `Timestamp` are set automatically at creation time and cannot be modified.
