# Class Sheet: IMessageEnvelope

**Namespace:** Carebed.Infrastructure.MessageEnvelope

---

## Purpose
Defines the contract for an envelope that wraps event messages in the Carebed system. Ensures consistency across modules and supports logging, debugging, resource cleanup, and extensibility.

---

## Interface Definition
```
public interface IMessageEnvelope : IDisposable
```

---

## Properties
- `Guid MessageId { get; }`
  - Unique identifier for the message envelope.

- `DateTime Timestamp { get; }`
  - Timestamp when the message was created.

- `MessageOrigins MessageOrigin { get; }`
  - The origin of the message (e.g., SensorManager, ActuatorManager).

- `MessageTypes MessageType { get; }`
  - Type of the message represented by this instance.

- `object Payload { get; }`
  - The actual payload of the message (non-generic access).
  - Required property (cannot be null).

---

## Methods
- `string ToString()`
  - Provides a string representation of the message envelope.

  - Releases resources used by the message envelope.
