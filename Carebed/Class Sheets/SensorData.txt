# Class Sheet: SensorData

**Namespace:** Carebed.Infrastructure.Message.SensorMessages

---

## Purpose
Represents a strongly-typed, immutable record for sensor data messages in the Carebed system. Used for transporting sensor readings and related metadata through the event bus.

public record SensorData

## Properties (All Required Init-Only)
- `double Value { get; init; }`
  - The measured value from the sensor (e.g., temperature reading, heart rate).
  - Required property.

- `string Source { get; init; }`
  - The unique identifier or logical name of the sensor (e.g., "tempSensor1", "RoomA_EEG").
  - Required property.

- `SensorTypes SensorType { get; init; }`
  - The type of sensor (e.g., Temperature, HeartRate, EEG).

- `bool IsCritical { get; init; }`
  - Indicates if the reading is considered critical (e.g., out of safe range).

## Usage Example
```csharp
var sensorData = new SensorData
{
    Value = 98.6,
    Source = "tempSensor1",
    SensorType = SensorTypes.Temperature,
    IsCritical = false,
    CreatedAt = DateTime.UtcNow,
    CorrelationId = Guid.NewGuid(),
    State = SensorStates.Active
};

var envelope = new MessageEnvelope<SensorData>(
    sensorData, 
    MessageOrigins.SensorManager, 
    MessageTypes.SensorData
);
```
